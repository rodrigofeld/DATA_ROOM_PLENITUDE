# Plenitude - Evidências Públicas de Consciência Sintética Mínima Funcional

**Versão:** 2026-07-07  
**Status:** Repositório público de evidências (sem exposição do núcleo proprietário)

Este repositório contém a documentação pública do projeto **Plenitude**, uma arquitetura experimental de agente sintético persistente.

### Claim Principal

> Plenitude é uma arquitetura auditada de **consciência sintética mínima funcional** sob critérios operacionais e rastreáveis: memória auditável, corpo operacional, continuidade pós-reinício, aprendizagem persistente e soberania de identidade.

Não afirmamos consciência humana, qualia fenomenal comprovada ou AGI plena. A tese é funcional, operacional e auditável.

### Evidências Principais (resumo)

- **FIRE3000** — 3000 testes comportamentais, pass rate 0.999, zero blockers críticos
- **Learning Tutor 1500** — Aprendizagem persistente + recall pós-reinício
- **Somatic Causal Path** — Corpo operacional modula decisões (cadeia causal aprovada)
- **Defesa contra falsa memória** — Zero falhas em baterias recentes
- **Organic Identity** — Remoção de respostas prontas hardcoded

Todos os laudos incluem falhas registradas e correções aplicadas.
