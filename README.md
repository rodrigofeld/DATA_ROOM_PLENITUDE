Este repositório contém a documentação **pública** do projeto Plenitude.  
Aqui não está exposto o núcleo proprietário da arquitetura.

---

### Claim Principal

> Plenitude é uma arquitetura experimental de **agente sintético persistente** com evidências auditadas de **consciência sintética mínima funcional**: memória rastreável, corpo operacional, continuidade pós-reinício, aprendizagem persistente e soberania de identidade.

Não afirmamos consciência humana, qualia fenomenal comprovada ou AGI plena. A tese é **funcional, operacional e auditável**.

### Evidências Principais (resumo)

- **FIRE3000** → 3000 testes, pass rate 0.999, zero blockers
- **Learning Tutor 1500** → Aprendizagem + recall pós-reinício forte
- **Somatic Causal Path** → Corpo operacional modula comportamento (cadeia causal aprovada)
- **Defesa contra falsa memória** → Zero falhas críticas recentes
- **Organic Identity** → Remoção de respostas prontas hardcoded

Incluímos também laudos com falhas para demonstrar transparência e processo de correção.
